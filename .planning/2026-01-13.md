## 프로젝트 분석 요약

### 프로젝트 정보
- **이름**: Google Nest Smart Home Experience
- **유형**: 스마트홈 체험 랜딩 페이지
- **목적**: Google Nest 제품 마케팅/인터랙티브 데모

### Tech Stack
- React 19.2.3 + TypeScript 5.8
- Vite 6.2 (빌드 도구)
- Framer Motion 12.25 (애니메이션)
- TailwindCSS (CDN)
- Google Gemini API (AI 음성 체험)
- Spline (3D iframe)

### 컴포넌트 (6개)
1. **Navigation** - 스크롤 반응형 헤더
2. **Hero** - 3D Spline 배경 + 타이틀
3. **ProductShowcase** - 제품 카드 그리드
4. **SmartZoneControl** - 3D 아이소메트릭 플로어플랜 (가장 복잡, 214줄)
5. **Estimator** - 스마트홈 견적 계산기
6. **VoiceExperience** - "오케이 구글" AI 음성 체험
