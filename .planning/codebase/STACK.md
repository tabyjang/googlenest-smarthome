# Tech Stack

## Languages
- **TypeScript** ~5.8.2 - 주 개발 언어
- **TSX** - React 컴포넌트 파일

## Frontend Framework
- **React** ^19.2.3 - UI 라이브러리 (최신 버전)
- **React DOM** ^19.2.3 - DOM 렌더링

## Build Tools
- **Vite** ^6.2.0 - 빌드 도구 및 개발 서버
- **@vitejs/plugin-react** ^5.0.0 - React 플러그인

## Styling
- **TailwindCSS** - CDN 방식 (script 태그로 로드)
- **Google Sans** 폰트 - 웹폰트

## Animation
- **Framer Motion** ^12.25.0 - 애니메이션 라이브러리

## AI/API
- **@google/genai** ^1.35.0 - Google Gemini AI SDK

## External Services
- **Spline** - 3D 인터랙티브 모델 (iframe 임베드)
- **Unsplash** - 이미지 소스
- **Picsum** - 플레이스홀더 이미지

## Development Dependencies
- **@types/node** ^22.14.0 - Node.js 타입 정의
- **typescript** ~5.8.2 - TypeScript 컴파일러

## Module System
- ES Modules (`"type": "module"`)
- Import maps (HTML 내 esm.sh CDN)

## Environment
- Node.js 필수
- `GEMINI_API_KEY` 환경 변수 필요 (.env.local)
