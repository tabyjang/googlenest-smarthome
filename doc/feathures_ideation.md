# 🚀 Google Nest Experience - 3대 킬러 기능 구현 가이드

**작성일:** 2026-01-14
**목표:** 방문자의 체류 시간을 늘리고, 실제 견적 요청(Conversion)으로 유도하는 핵심 기능 3종 구현.

## 1. 💰 우리 집 에너지 절약 계산기 (Energy ROI Calculator)

### 💡 기획 의도

단순히 제품 가격만 보여주는 것이 아니라, Nest Thermostat(온도조절기) 도입 시 **"언제 본전을 뽑는지(Break-even Point)"**를 시각적으로 보여주어 구매 명분을 만들어줌.

### 🛠️ 구현 로직 (Logic Flow)

1. **입력(Input):** 주거 형태(아파트/주택), 평수(Slider), 월평균 난방비(Number Input).
2. **계산(Process):**
    * 예상 절감액 = `월 난방비` × `0.12` (평균 12% 절감 가정)
    * 손익분기점(개월) = `기기값(약 30만원)` ÷ `예상 절감액`
3. **출력(Output):**
    * "고객님은 **8개월** 뒤부터 돈을 벌게 됩니다!" 문구 강조.
    * `Recharts` 라이브러리를 사용해 [기존 난방비 vs 절약된 난방비] 막대그래프 비교.

### 📦 필요 리소스 (Assets)

* `recharts` 라이브러리 (`npm install recharts`)
* 동전이 쌓이는 Lottie 애니메이션 또는 아이콘.

### ✅ Tasks Checklist

- [ ] **[UI]** 평수 조절 슬라이더(Range Input) 및 난방비 입력창 컴포넌트 제작.
* [ ] **[Logic]** 입력값에 따라 연간 절약액과 회수 기간을 계산하는 `calculateSavings` 함수 작성.
* [ ] **[Visual]** `Recharts`를 설치하고, 'Before & After' 데이터를 막대그래프로 시각화.
* [ ] **[Effect]** 결과 숫자가 `0`에서 `360,000`까지 촤르륵 올라가는 `CountUp` 애니메이션 적용.
* [ ] **[CTA]** 결과 하단에 "이 금액 절약하러 가기(견적 담기)" 버튼 연결.

## 2. 🎬 스마트홈 루틴 시나리오 (Routine Simulator)

### 💡 기획 의도

버튼 하나로 웹사이트 전체의 분위기(테마)를 바꾸며 스마트홈의 핵심인 **'자동화 경험'**을 웹에서 간접 체험하게 함.

### 🛠️ 구현 로직 (Logic Flow)

1. **상태 관리:** 전역 상태(Context API 또는 Zustand)로 `currentMode` 관리 (`default` | `cinema` | `sleep`).
2. **모드별 동작:**
    * **☀️ 굿모닝:** 배경 밝음, 커튼 열림(이미지 교체), 경쾌한 BGM(선택), Nest Hub에 날씨 표시.
    * **🍿 시네마:** 배경 어두움(Dark Mode), 주요 UI 숨김, Nest Audio에서 앰비언트 라이트 효과.
    * **🔒 외출(보안):** CCTV 화면 팝업, 도어락 잠김 아이콘 표시, "경비 모드 시작" 알림 토스트.

### 📦 필요 리소스 (Assets)

* 각 모드에 맞는 배경 이미지 3장 (낮, 밤, 거실).
* 제품 상태별 아이콘 (꺼진 전구 vs 켜진 전구).
* 효과음 파일 (선택 사항: 띠링~ 소리).

### ✅ Tasks Checklist

- [ ] **[State]** `ThemeContext.tsx`를 생성하여 `currentMode` 상태 전역 관리 설정.
* [ ] **[UI]** 화면 중앙 또는 하단에 3가지 모드(아침/영화/외출) 전환 토글 버튼 배치.
* [ ] **[Style]** Tailwind CSS의 다크 모드 클래스(`dark:`)를 활용해 시네마 모드 시 배경색 반전 처리.
* [ ] **[Interaction]** 모드 변경 시 해당 제품 아이콘(전구, 스피커)의 이미지/색상이 바뀌도록 조건부 렌더링 작성.
* [ ] **[Animation]** 배경 전환 시 `framer-motion`을 사용하여 부드럽게 페이드인/아웃 처리.

## 3. 🛋️ 가상 배치해 보기 (Virtual Room Planner - AR Lite)

### 💡 기획 의도

고객이 자신의 방 사진 위에 제품 스티커를 붙여보며 **"우리 집에 어울릴까?"**라는 고민을 해소하고 체류 시간을 극대화함.

### 🛠️ 구현 로직 (Logic Flow)

1. **캔버스 영역:** 배경 이미지를 까는 <div> 영역. (기본 이미지 제공 + 사용자 업로드 기능).
2. **스티커 덱:** 하단에 드래그 가능한 제품 아이콘 나열.
3. **드래그 앤 드롭:** `react-draggable` 라이브러리를 사용하여 아이콘을 캔버스 위 자유로운 위치로 이동.
4. **장바구니 연동:** 배치된 아이콘들의 리스트를 뽑아서 "이대로 견적 내기" 기능 연결.

### 📦 필요 리소스 (Assets)

* `react-draggable` 라이브러리 (`npm install react-draggable`)
* 배경이 투명한 제품 누끼 이미지(PNG) - Nest Audio, Hub, Thermostat 등.
* 기본 거실/침실 예시 이미지 2~3장.

### ✅ Tasks Checklist

- [ ] **[Setup]** `react-draggable` 라이브러리 설치.
* [ ] **[UI]** 화면을 위(캔버스)와 아래(제품 목록)로 나누는 레이아웃 잡기.
* [ ] **[Feature]** `<Draggable>` 컴포넌트로 제품 이미지를 감싸서 드래그 기능 구현.
* [ ] **[Feature]** `<input type="file">`을 이용해 사용자가 자기 방 사진을 배경으로 깔 수 있게 구현.
* [ ] **[Logic]** 캔버스 위에 올라간 제품들의 ID를 배열(`placedItems`)로 저장하는 로직 작성.
* [ ] **[CTA]** "이 배치대로 견적 요청하기" 버튼 클릭 시, 저장된 배열을 견적 페이지로 넘기기.
